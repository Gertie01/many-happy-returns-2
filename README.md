# Riffusion Studio - Model FUZZ-0.8

This project provides a fully functional interface for music generation using the **FUZZ-0.8** model. 

## Features
- **Endless Predictions**: No daily limits.
- **Dynamic Randomizer**: Automatic song duration calculation (30s-240s).
- **Advanced Editing**: Crop, Extend, Swap Vocals, and Generate Covers.
- **Library**: Local storage of generations.

## Usage
Run `python app.py` to launch the Gradio interface.
